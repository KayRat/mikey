# mikey
A truly unique admin mod for Garry's Mod, named after Michael Scott, star of The Office.

## data/mikey_config.json
This file is necessary to establish a database connection (aka how the mod stores data).

````json
{
  "hostname"  : "",
  "username"  : "",
  "password"  : "",
  "database"  : "",
  "port"      : 3306
}
````
